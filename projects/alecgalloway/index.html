Alec Galloway Portfolio

HTML, CSS, JQuery

This site contains a catalogue of world-renowned artist Alec Galloway's fine art and stained glass. As instructed, this site contains a lot of high resolution images on each page for an immersive feeling. I wanted this site to have a style that compliments that of Alec's work, so the font and colour scheme were carefully chosen to achieve this.

Features inlude:

  • Jquery Features:
    • Fading Navbar
    • Gallery Filters
    • Custom Scrollwheel
    • Gallery Image Modals

Site can be viewed live here - http://alecgalloway.net/
